# @stillpoint/desktop

टॉरी v2 का डेस्कटॉप रैपर, जो स्टिलपॉइंट मिक्सर को एक देशी विंडोज एप्लिकेशन में शामिल करता है। यह रिएक्ट फ्रंटएंड को लोकलहोस्ट पर चलने वाले नोड.जेएस सर्वर से जोड़ता है।

## विशेषताएं

- देशी विंडो (700x500, न्यूनतम 500x400)
- एनएसआईएस और एमएसआई इंस्टॉलर विकल्प
- सीएसपी केवल लोकलहोस्ट कनेक्शन तक सीमित

## चलाने के लिए

```bash
cd apps/desktop
npm run tauri dev
```

इसके लिए पोर्ट 3456 पर चलने वाला सर्वर और पोर्ट 5177 पर चलने वाला यूआई डेवलपमेंट सर्वर आवश्यक है।
